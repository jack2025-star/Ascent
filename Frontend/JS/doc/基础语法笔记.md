# 基础

## 📚 JavaScript 核心基础知识

### 一、变量声明

- **const**：声明的变量不可修改，是现代 JS 开发中的首选。
- **let**：声明的变量可以修改，在值需要发生变化时使用。
- **var**：旧语法，存在变量提升和作用域等问题，避免使用。

### 二、数据类型

- **string**：文本类型，例如 `"Hello"`。
- **number**：数字类型，例如 `30`。
- **boolean**：布尔类型，表示真或假，例如 `true`。
- **undefined**：表示变量已声明但未被赋值。
- **null**：表示空值，通常用于主动清空对象引用。

### 三、运算符

- **数学运算符**：`+` (加)、`-` (减)、`*` (乘)、`/` (除)、`**` (幂)。
- **比较运算符**：`>` (大于)、`<` (小于)、`>=` (大于等于)、`<=` (小于等于)。
- **赋值运算符**：`=` (赋值)、`+=`、`-=`、`*=`、`/=`。

### 四、字符串

- **拼接**：使用加号连接，例如 `"I'm " + name`。
- **模板字面量**：使用反引号和 `${}` 进行插值，例如 ``I'm ${name}``（推荐写法，可读性更强）。

### 五、条件语句

- **if/else**：用于基本的逻辑判断。
- **else if**：用于处理多个条件的判断。
- **switch**：适用于同一变量的多分支选择结构。
- **三元运算符**：简洁的条件表达式，语法为 `条件 ? 真值 : 假值`。

### 六、类型转换

- **显式转换**：手动调用函数进行转换，如 `Number()`、`String()`、`Boolean()`。
- **隐式转换**：JS 引擎自动进行的类型转换，例如 `"23" - "10"` 会自动转为数字并得出 `13`。

### 七、Falsy 值（虚值）

在布尔上下文中会被认定为 `false` 的值：

- `0`（以及 `-0`）
- `""`（空字符串）
- `undefined`
- `null`
- `NaN`（非数值）

### 八、相等运算符

- **==**：宽松相等，比较时会先进行隐式类型转换。
- **===**：严格相等，不进行类型转换，要求类型和值完全一致（推荐使用）。

### 九、逻辑运算符

- **&& (AND)**：逻辑与，所有条件都为真时才为真（常用于短路求值）。
- **|| (OR)**：逻辑或，只要有一个条件为真就为真（常用于设置默认值）。
- **! (NOT)**：逻辑非，对布尔值进行取反。




### 严格模式 (Strict Mode)

> 在文件最顶部加 'use strict';
>
> 作用：让 JS 更严格检查代码，提前暴露错误
>
> 例：未声明变量会报错，保留字不能当变量名



 ###   函数 (Functions)

1. 函数声明
> 用 function 关键字
>
> 例：`function logger() { console.log('Jack'); }`
>
> 调用：logger()

2. 函数表达式
>  把函数赋值给变量
>
> 例：`const calcAge = function(birthYear) { return 2026 - birthYear; }`
>
> 必须先定义后调用

3. 箭头函数
> 用 => 箭头符号，更简洁
>
> 单行自动返回：const calcAge2 = birthYear => 2026 - birthYear
>
> 多行需大括号和 return：const add2 = (a, b) => { return a + b; }

> 参数规则：
>
> 0 个参数：() => ...
>
> 1 个参数：x => ...
>
> 多个参数：(x, y) => ...

4. 参数与返回值

   	>参数：函数的输入（如 apples, oranges）
   	>
   	>return：函数的输出（如 return juice）


5. 函数调用函数
> 函数内部可以调用其他函数
>
> 例：fruitProcessor 内部调用 cutFruitPieces
>
> 作用：代码更模块化，每个函数只做一件事
